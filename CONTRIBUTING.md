# Contributing

## Development

- Install dependencies with `pnpm install --ignore-scripts`.
- Run checks with `CI=true pnpm test`.
- Build with `pnpm build`.
- Check for dead code with `pnpm knip`.

## Clever Cloud E2E

Maintainer-only live E2E setup and repair live in [`docs/e2e-operations.md`](docs/e2e-operations.md).
User-facing action docs stay in `README.md`.
